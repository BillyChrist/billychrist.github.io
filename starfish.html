STARFISH

Soft Translatable Actuated Robot for in Space Handling

SERC soft robotics research project characterizing Shape Memory Alloy (SMA) actuators and limb motion using custom motion capture and data analysis tools.

Research Goals

This is a stand-in paragraph, content to follow!

  • Quantify SMA actuation uncertainty across multiple thermal cycles.
  • Develop testbeds for 1D and 2D limb characterization using motion capture and thermocouples.
  • Automate data collection and analysis for temperature, strain, and bend angle.

SMA Preparation

This is a stand-in paragraph, content to follow!

  • Prepared thermocouples using Kapton tape and thermal adhesive to minimize added thermal mass.
  • Designed attachment method to avoid electrical conduction between SMA and thermocouples.

1D SMA Characterization

This is a stand-in paragraph, content to follow!

  • Built 1D actuation stand with calibrated camera and motion capture target.
  • Ran multiple trials across voltage and mass combinations to measure contraction/relaxation times.
  • Reported averages such as 5.7 ± 0.7 s contraction and 27 ± 2 s relaxation for representative cases.

2D Limb Characterization

This is a stand-in paragraph, content to follow!

  • Created top-down camera setup with multi-color motion capture markers on a full limb.
  • Automated experiments for target temperature vs. bend angle, bend angle vs. time, and actuation time vs. angle.
  • Analyzed uncertainty ranges during primary actuation and improved thermal/position models.

Software

This is a stand-in paragraph, content to follow!

  • Developed motion capture pipeline using OpenCV with contour moments for accurate ball tracking.
  • Computed bend angles and arc radii from marker positions with color-separated tracks.
  • Produced averaged measurements for more reliable series data across trials.
